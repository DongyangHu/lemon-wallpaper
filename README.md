## 简介
柠檬壁纸基于WinFrom开发，实现每日自动更换壁纸的能力。主要功能如下：
- 每日自动下载壁纸文件并更新桌面壁纸
- 壁纸文件自动清理
- 历史壁纸管理

|选项|说明|
|:--------:|:--------:|
|开发语言|C#|
|.Net版本|V4.7.2|
|IDE|Visual Studio 2022|
